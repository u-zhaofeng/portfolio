Grow and Roll

Second game development project - a familar battleground, but still poses a challenge. A very good coding and problem solving challenge indeed.

Project Overview

As I thoroughly enjoyed the process of making my first game Wrecking Draw, I decided to come up with another original idea for a game. Thats when I came up with the concept of a rolling ball that has to change its size to get past various obstacles. Even though at this point I has some prior experience with C# and Unity, I still found this project challenging as it was a another genre of game completely and had more moving objects than what I've worked with before.

I started this project as I knew it was a good software development challenge to test and improve both my coding and problem solving skills. Being resourceful was also crucial as some of the concepts involved in this project was really niche and solutions was few and far between.
