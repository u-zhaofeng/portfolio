Do Nothing Box

This project marks my first of many steps into hardware development, or more specifically Arduino development.

Project Overview

One day, I saw a similar machine showcased in a YouTube video and I started wondering if I could do it myself. Of course, I wanted to challenge myself so I gave it a try. With zero knowledge on how to even start, this project was quite the challenge. However after months of on and off work (I got the components from China and it took more than 2 weeks to ship), I successfully made it work, and could not be any prouder.

While completing this project, I learned many new skills and refined existing ones. I will most definitely be working on more Arduino based projects soon, where the valuable experience obtained from this project should be useful.

Note: If you are visiting this page when the box box looks incomplete with 5 switches (supposed to have 8), its because the remaining switches have not arrived yet, I will update when I fully complete it.
