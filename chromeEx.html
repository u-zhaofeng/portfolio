Custom Week Number - Chrome Extension

Struggled with a problem on Google calendar, and challenged myself to find a solution in a form of an extension.

Project Overview

When I was going through the semesters in school, I used Google Calendar extensively to manage my busy schedule. However, I quickly realised that I could not change the week number to correspond to my school week. After searching relentlessly in vain for solutions, I decided to create my own solution in the form of a Chrome Extension, not only to help myself, but also to help others in the same plight.

The biggest challenge I faced when working on this project was understanding Google Calendar's native HTML and JavaScript code, so that I could modify them effectively. In hindsight, I realised how important this skill is as rarely in software develpment do one get to start from scratch. This project definitely helped me work on this skill.
