Custom Week Number for Google Calendar

Project Overview

When I was going through the semesters in school, I used Google Calendar extensively, mainly to manage my schedule. However, there was no option to change the week number to one that corresponded to my school week. I decided to create an extension to do just this, not only to help myself in the future, but also to help others who might need it.

One thing I learnt while making this extension was how to understand other's code. Since this is an extension, I do not have the privilege to change the original code (unlike previously when I was working with my own code), and hence I have understand the already written code and figure out a way to build on it. I think this skill should help me work well with other's better since I can't have it my way all the time.
