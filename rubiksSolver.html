Rubik's Cube Analyser and Solver

This challenging project was sparked by a theory I had about automating the process of solving a Rubik's cube without any user input.

Note: In the video, the solution printed out is in standard Rubik's Cube notation

Project Overview

Before I learnt how to solve a Rubik's cube, I used a tool online that generated the solution for any configuration of a cube. Only problem is that I have to input all 6 faces one by one. Ever since then, I've always wondered if it was possible to read all faces of a Rubik's cube using software and generate a solution with no user input at all. I expanded on this idea and eventually realised that if I rotate the cube at one of its corners, a camera set at a fixed point could capture all faces of the cube. I decided to put this theory to the test by developing and assembling both software and hardware, and eventually, I managed to achieve this.

Working with machine learning was challenging to say the least, requiring extensive troubleshooting to deal with inaccurate detections. After having to train and fine-tune my own YOLOv8 model to detect the corners of the cube, I completed this project with a greater appreciation for machine learning and its potential.

A big part of this project relied on image processing, being able to tell the colors apart in inconsistent lighting conditions. After attempting many solutions for this problem, I can safely say my knowledge and understanding of image processing has greatly improved.
